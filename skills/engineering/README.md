# engineering

Skills for day-to-day code and backend work.

- **[coderabbit-request](./coderabbit-request/SKILL.md)** — After finishing a change, request a CodeRabbit review of your uncommitted diff and return a structured, severity-categorized (critical/important/minor) issue list to drive your fix workflow.
- **[convex-domain-folder](./convex-domain-folder/SKILL.md)** — Reorganize a Convex backend into per-domain folders (each domain owns its `schema.ts`/`queries.ts`/`mutations.ts`/`model.ts`), correctly repointing the `api.<domain>.*` function paths and co-located test resolution.
